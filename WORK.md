# JLPT N2 한자 읽기 암기 앱

## 프로젝트 개요
- **목표:** JLPT N2 단어의 한자 읽기(히라가나)를 암기하는 iOS 앱
- **타겟:** 한자 읽기에 어려움을 겪는 한국어 사용자
- **디자인:** 미니멀
- **플랫폼:** iOS only (Flutter)

## 데이터 준비

### 원본 데이터
- **출처:** [jlpt-vocab-api](https://github.com/wkei/jlpt-vocab-api) (REST API)
- **API:** `https://jlpt-vocab-api.vercel.app/api/words?level=2`
- **총 단어 수:** 1,831개
- **제공 필드:** word(한자), furigana(히라가나), meaning(영어), romaji, level

### 데이터 가공 파이프라인

| 단계 | 작업 | 상태 |
|------|------|------|
| 1 | API에서 N2 전체 단어 JSON 다운로드 | [ ] |
| 2 | 영어 뜻 → 한국어 번역 | [ ] |
| 3 | 예문 추가 (일본어 예문 + 한국어 해석) | [ ] |
| 4 | furigana 빈 값 보완 (카타카나/히라가나 단어) | [ ] |
| 5 | 최종 데이터 검증 및 앱 번들용 JSON 생성 | [ ] |

### 최종 데이터 스키마
```json
{
  "word": "題名",
  "reading": "だいめい",
  "meaning": "제목",
  "pos": "명사",
  "example": {
    "ja": "この本の題名は何ですか。",
    "reading": "このほんのだいめいはなんですか。",
    "ko": "이 책의 제목은 무엇입니까?"
  }
}
```

### 한국어 번역 전략
- **방법:** 영어 뜻을 기반으로 번역 API 활용 또는 수동 검수
- **주의:** 일본어→한국어 직역이 더 정확하므로, 가능하면 일본어 원문 기준 번역
- **검수:** 자동 번역 후 수동 검수 필요 (특히 다의어)

### 예문 전략
- **출처 후보:** Tatoeba 코퍼스 (CC 라이선스), 또는 LLM 생성
- **구성:** 일본어 예문 + 히라가나 읽기 + 한국어 해석
- **기준:** N2 수준에 맞는 난이도, 해당 단어의 용법을 잘 보여주는 문장

## 앱 기능 (우선순위순)

### Phase 1 - MVP
1. **플래시카드** — 한자 → 히라가나 읽기 + 한국어 뜻
2. **퀴즈** — 한자 보고 히라가나 4지선다
3. **단어장** — 전체 리스트 검색/필터

### Phase 2
4. **간격 반복 (SR)** — SM-2 알고리즘 기반 복습 스케줄
5. **학습 통계** — 진도율, 정답률

### Phase 3
6. **예문 학습** — 예문 속 한자 읽기 연습
7. **북마크** — 어려운 단어 모아보기

## 기술 스택
- **Framework:** Flutter (iOS only)
- **로컬 DB:** SQLite (sqflite) 또는 Hive
- **상태관리:** Riverpod
- **데이터:** 앱 번들 JSON (assets)
