# JLPT N2 한자 읽기 암기 앱

JLPT N2 단어의 한자 읽기(히라가나)를 암기하기 위한 iOS 앱입니다.

## 주요 기능 (예정)
- 플래시카드 — 한자를 보고 히라가나 읽기 학습
- 퀴즈 — 한자 보고 히라가나 4지선다
- 단어장 — 전체 N2 단어 리스트 검색/필터
- 예문 학습 — 예문 속 한자 읽기 연습

## 데이터

- **구성:** 한자, 히라가나 읽기, 한국어 뜻, 예문(일본어 + 한국어 해석)

## 기술 스택

- Dart
